# Bases Angular

Proyecto para conocer las bases de angular
Algunos conceptos son:
    - componentes
    - templetes html y css
    - interpolatión (<p>{{propiedad}}</p>)
    - exportación de módulos
    - directivas:
         *ngIf  
         *ngFor
    - eventos (click)=""


CLI Commandos: 
    - ng new my-first-project
    - cd my-first-project
    - ng serve --open

Generete component: 
    - ng generate component [name]
